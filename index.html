Travisified

Pinging when Travis builds finish in a GitHub Pull Request.

Step 1.
Add the following to your .travis.yml to play ball.

notifications:
  webhooks: https://travisified.paas.allizom.org/webbook

Step 2.
Make a pull request to projects.ini to include your project's config there.

Step 3.
Run Travis CI Jobs and hope for the best!
